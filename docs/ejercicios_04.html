Ejercicios de Clase 04

  1. Escriba una función que tome como un argumento una cadena de caracteres (string) y lo devuelva centrado con un ancho w. Si la longitud del string es mayor que el ancho w debe retornar el string original. Los argumentos que toma la función son el string y el ancho.

  2. Utilizando conjuntos (set), escriba una función que compruebe si un string contiene todas las vocales. La función debe devolver True o False.
